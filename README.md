# EntregaFinal_IndaPablo

En este repositorio se encuentra el Proyecto final del Curso de SQL dictado por CODERHOUSE

Aqui hay dos archivos y un directorio.

El archivo Eventos_Artisticos_IndaPablo.pdf es la documentacion del proyecto de Base de Datos llamada Eventos Artisticos.
El cual contienete toda la descripcion de la DB, para poder comprender su objetivo, funcionamiento y uso.

El archivo Instructivo de Instalacion, es la guia para instalar la Base de Datos.

El directorio Eventos_Artisticos_Files contiene todos los archivos necesarios para la instalacion, los cuales estan referenciados en el archivo de Instalacion.
